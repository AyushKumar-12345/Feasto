# 🍽️ Feasto Admin Dashboard

The **Feasto Admin Dashboard** is a high-performance, modern administration suite for the Feasto food delivery platform. It enables administrators to manage the restaurant menu, oversee customer orders, monitor payment status, and manage the complete order lifecycle through a responsive and intuitive interface.

## 🚀 Features

* **Dynamic Menu Management:** Add food items with Cloudinary image uploads and automatic image optimization.
* **Inventory Management:** View and remove food items with real-time updates.
* **Order Management:** Monitor customer orders, delivery details, payment status, and purchased items.
* **Order Status Tracking:** Update order progress from **Food Processing → Out for Delivery → Delivered**.
* **Payment Management:** Securely verifies Razorpay payments and maintains payment status for every order.
* **Cloudinary Integration:** Fast and optimized cloud-based image storage.
* **JWT Authentication:** Secure admin login and protected API access.
* **Responsive Dashboard:** Clean and responsive interface optimized for desktop and mobile devices.

## 🛠️ Tech Stack

* **Frontend:** React 19, Vite, React Router DOM v7
* **HTTP Client:** Axios
* **Icons:** Lucide React
* **Notifications:** React Toastify
* **Styling:** Pure CSS
